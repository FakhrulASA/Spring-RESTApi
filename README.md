## Create RESTful API with Spring Boot in kotlin language.

### Here the following tasks have been implemented,

1. Creating Layers inluding data, web, service, controller, mock
2. POST, GET, PATCH, DELETE and FETCH API.
3. Junit 5 Testing
4. Response handling
5. Mock Datasource
